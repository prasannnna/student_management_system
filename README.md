# Student Management System – Java

## Overview

Student Management System is a simple Java console-based application developed to manage student records efficiently. It allows users to perform basic CRUD operations such as adding, viewing, searching, updating, and deleting student details.

This project is ideal for beginners to understand Java programming concepts like Object-Oriented Programming (OOP), Collections, Exception Handling, Loops, and Methods.

---

## Features

* Add new student records
* View all student records
* Search student by ID
* Update student details
* Delete student records
* User-friendly menu-driven console interface

---

## How to Run the Project

### Step 1: Save the File

Save the code as:

```text
StudentManagementSystem.java
```

### Step 2: Compile the Program

```bash
javac StudentManagementSystem.java
```

### Step 3: Run the Program

```bash
java StudentManagementSystem
```
